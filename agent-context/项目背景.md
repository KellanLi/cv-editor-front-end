# AI 简历编辑系统（毕设）

面向毕业生的简历编辑产品：用可复用的**模块模版（Section Template）**与**信息层（Info Layer）**拼装经历类区块，并配合 AI 能力辅助编辑（具体能力随业务迭代补充）。

## 技术栈

Next.js（App Router）、pnpm、HeroUI。

## 功能范围（概要）

- 登录 / 注册  
- **简历**：按简历维度的增删改查  
- **模块**：按 Section / 模版维度的增删改查（与「信息层」拼装规则见下方引用）

## 领域模型（简要）

简历 → 多个 **Section**；每类区块对应一种 **Section Template**。模版由三种 **Info Layer**（左标题右时间段、左右文字、富文本）组合而成，用户为各层字段配置展示名称。

完整术语表、目录约定与易混产品事实见 **`.cursor/rules/project.mdc`**（默认注入，无需重复粘贴）。
