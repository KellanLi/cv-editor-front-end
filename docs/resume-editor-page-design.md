# 简历编辑页面设计

## 整体设计

整体布局为：顶部有导航栏，下面是“左-中-右”三栏布局，左栏和右栏可以收起/展开

- 左栏为AI对话区
- 中部为简历编辑/预览区
- 右侧为多功能区，包含：模块管理、设计、AI诊断

## 导航栏设计

顶部的导航栏分为靠左和靠右两个区域：

- 靠左依次为：返回简历列表页的返回按钮、简历名称、简历状态（同步中/已保存）
- 靠右依次为：控制左栏收起/展开的按钮，控制右栏收起/展开的按钮，导出按钮，手动保存按钮

## AI对话区设计

暂无，目前先不做开发

## 简历编辑/预览区设计

顶部固定为个人信息模块，后面为可改变顺序的自定义模块。

**个人信息模块**

至少包含：

1. 证件照
2. 姓名
3. 出生日期
4. 性别
5. 目标岗位
6. 联系邮箱
7. 联系电话

同时可以自行增加标签。

和section-render一样都有两种状态：edit和view。点击模块区域即可进入edit状态。

在edit状态下：点击证件照块即可弹出证件照上传弹出框。其余至少包含的信息均为输入框状态，可以通过输入修改信息。在末尾，可以通过合理的交互进行添加自定义标签的操作。

在view状态下：如果字段有值则渲染，无值或为空则不渲染对应字段。

**自定义模块**

基于src/components/section-render.tsx继续实现。

点击它渲染出的前端层，前端层可以转换为active状态。

如果其中有active状态的前端层。这个模块为edit状态。

## 多功能区设计

顶部为三栏（模块管理、设计、AI诊断）导航栏。

**模块管理**

- 有添加模块按钮
- 有简历中包含有的模块列表，模块列表中模块的顺序和简历中模块顺序一一对应
- 拖拽列表中的模块可以更改简历中模块的顺序
- 点击模块列表中的模块可以在简历编辑区定位到对应模块
- 可在列表中删除模块
